Weather App

A simple, reliable weather app! Search any town or city in the world and get current weather information, as well as a few recent news stories. This app accesses the OpenWeather API to retrieve weather and time data for any given location, as well as the World News API to retrieve news related to the searched location. I also built a proxy server using Express and Node so that the API call as well as my personal API key (for OpenWeather) are hidden from the client-side. Disclaimer: OpenWeather doesn't seem to recognize a lot of obscure locations, so some smaller towns may not return any data.
